HOMESTAY PDPA BOOKING V4 — SETUP GOOGLE SHEETS + GOOGLE DRIVE

1. CIPTA GOOGLE SHEET
- Buka Google Sheets dan cipta spreadsheet baru.
- Salin Spreadsheet ID dari URL: docs.google.com/spreadsheets/d/ID_DI_SINI/edit

2. CIPTA FOLDER GOOGLE DRIVE
- Cipta satu folder khas, contoh: Homestay Booking IC & Signature
- Salin Folder ID dari URL folder Drive.
- PENTING: Jangan set folder sebagai public. Hadkan akses kepada owner/admin sahaja.

3. SETUP GOOGLE APPS SCRIPT
- Buka script.google.com dan cipta New Project.
- Copy semua kod dari fail google-apps-script.gs ke Code.gs.
- Tukar:
  SPREADSHEET_ID = 'PASTE_SPREADSHEET_ID_HERE'
  DRIVE_FOLDER_ID = 'PASTE_DRIVE_FOLDER_ID_HERE'
- Project Settings > Time zone > pilih (GMT+08:00) Kuala Lumpur.

4. DEPLOY WEB APP
- Klik Deploy > New deployment > Web app.
- Execute as: Me.
- Who has access: Anyone.
- Klik Deploy dan beri permission.
- Copy URL Web App yang berakhir dengan /exec.

5. LINK WEBSITE
- Buka script.js.
- Cari WEB_APP_URL.
- Gantikan PASTE_YOUR_GOOGLE_APPS_SCRIPT_WEB_APP_URL_HERE dengan URL /exec tadi.

6. HOST WEBSITE MELALUI HTTPS
- Upload index.html, style.css, script.js ke hosting HTTPS.
- Kamera browser memerlukan HTTPS atau localhost.

7. TEST
- Isi booking ujian.
- Ambil gambar IC depan/belakang.
- Tick PDPA dan sign.
- Submit.
- Semak Google Sheet: data booking mesti muncul.
- Semak Google Drive: 3 fail mesti muncul (IC depan, IC belakang, tandatangan).
- Cuba booking tarikh overlap: server akan reject.

NOTA KESELAMATAN
- Gambar IC ialah data peribadi berisiko tinggi. Jangan jadikan folder Drive public.
- Hadkan akses hanya kepada staf yang perlu.
- Tetapkan tempoh simpanan dan proses pemadaman.
- Consent ringkas dalam demo bukan pengganti nasihat undang-undang atau notis privasi lengkap.
